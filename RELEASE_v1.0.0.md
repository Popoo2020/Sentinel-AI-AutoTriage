# Release v1.0.0 - Sentinel-AI-AutoTriage

🚀 **Initial Release**

## Overview
Cloud-native SOC automation framework that connects to Microsoft Sentinel, ingests Tier-1 incidents, and uses LLM APIs to analyze, enrich, and auto-close incidents.

## Key Features
- Automated incident ingestion from Microsoft Sentinel
- LLM-powered incident analysis and enrichment
- Intelligent auto-closure of low-risk incidents
- Cloud-native architecture

This is part of a comprehensive security & AI automation platform alongside other enterprise-grade tools.
