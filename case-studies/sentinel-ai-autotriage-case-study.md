# Sentinel AI AutoTriage - Case Study

## Problem

Security teams need faster ways to review repetitive incident information while keeping accountability and review controls visible.

## Solution

This project demonstrates a Microsoft Sentinel-oriented portfolio workflow with:

- pre-review redaction,
- structured recommendations,
- deterministic policy checks,
- approval state for sensitive workflow paths,
- and metadata-only audit records.

## Outcome

The project shows how AI-assisted security operations can be presented with governance, reviewability and safety boundaries rather than uncontrolled automation.

## Portfolio value

Relevant for AI Security, SOC automation, Microsoft Sentinel, cybersecurity GRC and secure-by-design automation roles.
